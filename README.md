# 🚀 Nebula — SaaS Landing Page

A landing page for a fictional analytics startup. Looks like the website of a real, funded company.

**▶ [Live demo](https://xdxdxdcoder.github.io/nebula-landing/)**

![vanilla](https://img.shields.io/badge/JS-vanilla-yellow) ![responsive](https://img.shields.io/badge/responsive-yes-brightgreen) ![no build](https://img.shields.io/badge/build-none-blue)

## Features
- ✨ Animated hero with a glow effect and a "live" mock dashboard
- 🔀 **Monthly / yearly** pricing toggle with recalculated prices
- 📜 Scroll-reveal animations (`IntersectionObserver`)
- ❓ FAQ accordion
- 📱 Fully responsive with a mobile burger menu
- 🎨 Sticky navigation, gradient text, section-by-section composition

## Run
```bash
open index.html
```

## Tech
Vanilla JS · CSS Grid / Flexbox · `IntersectionObserver` · CSS animations · responsive layout
